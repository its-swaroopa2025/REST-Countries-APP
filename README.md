# 🌍 Rest Countries Explorer

A responsive web application that allows users to explore information about countries using the REST Countries API. This project demonstrates API handling, dynamic UI rendering, and modern frontend features like dark/light mode.

---

## 🚀 Features

* 🌍 Fetch and display countries using REST Countries API
* 🔍 Search countries by name
* 🌐 Filter countries by region
* 📄 Separate country detail page
* 🔗 Click on border countries to navigate to their details
* 🌙 Dark / Light mode toggle
*   ❌ Displays **"Country not found"** for invalid searches
* 🚫 Shows **"No border countries"** message when applicable

---


## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)
* REST Countries API
